# 📱 Ketoclub App

App móvil exclusiva para miembros de la comunidad Ketoclub.

## Requisitos
- Node.js instalado (nodejs.org)
- Expo CLI: `npm install -g expo-cli`
- App "Expo Go" en tu celular Android (Play Store)

## Cómo correrla

1. Abrí la terminal en esta carpeta
2. Instalá dependencias:
   ```
   npm install
   ```
3. Iniciá la app:
   ```
   npx expo start
   ```
4. Escaneá el código QR con la app Expo Go en tu celular

## Pantallas incluidas
- ✅ Login con número de WhatsApp (verifica contra Supabase)
- ✅ Home con desafío del mes, reflexión del día y accesos rápidos
- 🔜 Recetas keto
- 🔜 Comunidad y chat
- 🔜 Ebooks y lives
- 🔜 Mi progreso
- 🔜 Desafíos

## Tecnología
- React Native + Expo
- Supabase (misma base de datos del panel web)
